# Xymon Wiki

Collaborative, community-maintained knowledge base for the Xymon ecosystem — a single,
searchable home for HOWTOs, troubleshooting, platform notes and client documentation,
to reduce repetition and scattered guidance.

## What is Xymon?

Xymon is a system for monitoring servers and networks. Inspired by Big Brother, it scales
from a handful of hosts to thousands of servers and services: a central server checks
network services (HTTP, FTP, SMTP, …) and — through agents — local disk, logfiles and
processes, builds status web pages with history and trend graphs, and raises alerts by
e-mail, SMS or pager. See it live at **[xymon.com](https://www.xymon.com/)** or read
[About Xymon](https://www.xymon.com/help/about.html).
