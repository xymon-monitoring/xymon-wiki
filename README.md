# xymon-wiki

Collaborative knowledge base for the Xymon monitoring ecosystem.

### 📖 Read it online → **<https://xymon-monitoring.github.io/xymon-wiki/>**

The content lives as Markdown under [`docs/`](docs/) and is published automatically
(MkDocs Material → GitHub Pages) on every push to `main`.

## Editing

- **With write access** — click the ✎ pencil at the top of any page; it opens the
  GitHub editor. Commit to `main` and the site republishes automatically.
- **Without write access** — the same pencil lets you propose a change as a pull
  request (GitHub forks the repo for you), or
  [open an issue](https://github.com/xymon-monitoring/xymon-wiki/issues) to suggest
  or report something.
